# NStatistics Plugin

Statistics plugin for PocketMine-MP servers (PlaceholderAPI).

---

## 📦 Dependencies

This plugin depends on the following libraries:

* **Commando** by CortexPE
  https://github.com/CortexPE/Commando

* **SimplePacketHandler** by Muqsit
  https://github.com/Muqsit/SimplePacketHandler

* **PlaceholderAPI** by NetherByte233
  https://github.com/NetherByte233/PlaceholderAPI

Make sure these dependencies are installed or included as virions before running the plugin.
